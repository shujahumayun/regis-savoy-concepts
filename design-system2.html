The rendered companion to design2.md. Deep midnight and warm ivory, one considered gold accent, a grotesque display paired with a serif reserved for figures and a monospace "ledger" for all notation. Structural, ruled, precise — the register of a prospectus.

Rules

  • Headlines: Archivo, sentence case, weight 500–600, tight tracking (-.02em). No terminal period. Never all-caps.
  • Markers / eyebrows: IBM Plex Mono, uppercase, wide tracking, slate grey, led by a gold FIG. 0X index and a short gold hairline.
  • Figures: Spectral, tabular (tnum), for stat numerals and large data only — with a small mono unit in gold.
  • Body: Archivo 400–500, sentence case, on a softened tier (never pure black/white on its ground). strong lifts to full midnight/ivory.

Rhythm

Motion

  • Transforms & opacity only — no animated filters, blur, noise, or marquees.
  • Rule draw-in: section hairlines and the register tick draw with scaleX/scaleY from the edge on scroll.
  • Reveal: sections fade + rise (IntersectionObserver, .reveal → .in), .8s ease, staggered .08/.16/.24s.
  • Hover: buttons lift 2px; links grow a gold underline; index rows slide right + icon golds; portraits zoom 1.04.
  • All motion gated behind a .js class and disabled under prefers-reduced-motion.

FIG. 08 Don'ts & Constraints

The guardrails

Brand / visual

  • Never a boxed monogram.
  • Never a generic purple-to-blue "AI" gradient.
  • Never fill large areas with gold — it's a rule / register-mark / mono-numeral / single-CTA accent.
  • Never more than the one gold accent at weight per composition.
  • Never a terminal period on headlines, and never all-caps headlines.
  • Never fake urgency, promise returns, or crowd the page with claims.

Technical (carry into the Wix build)

  • Self-contained — one HTML file per page, CSS/JS inline. No external JS libraries.
  • Google Fonts is the only external runtime resource.
  • Transforms & opacity only for animation — no marquees / animated filters.
  • Works fully without JavaScript — JS only enhances.
  • Mobile-first, smooth on low-end laptops; respect reduced-motion.
